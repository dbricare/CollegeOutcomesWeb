Project Description

Ideally, prospective students would like to maximize their earnings for their chosen tuition price range. There are two important factors to consider, first the dollar amount they can expect to earn and second how likely they are to meet that dollar amount.

Additional analysis has shown alumni earnings have a moderate positive correlation with university tuition. See the project details section for more information.

Project Details

  • Full project analysis on GitHub

  • Tools used include PostgreSQL, Python (sqlalchemy, pandas) libraries
  • Data is from the US Dept. of Education College Scorecard
  • Data spans years from 1996 to 2015
